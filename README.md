TechSpot - Sitio Web de Tienda de Electrónicos

TechSpot es un proyecto educativo sobre una tienda de electrónicos que busca expandir su presencia en línea mediante la creación de un sitio web interactivo donde los clientes puedan explorar y comprar una amplia gama de productos electrónicos, desde dispositivos móviles hasta equipos de entretenimiento doméstico.

Descripción del Proyecto


Este proyecto tiene como objetivo desarrollar un sitio web funcional y atractivo que cumpla con las necesidades y expectativas de los clientes de TechSpot. El sitio incluye elementos básicos de aplicaciones web, como formularios HTML5, integración de estilos con CSS y diseño adaptativo para asegurar una experiencia de usuario óptima en diferentes dispositivos.
Características Principales

1. Diseño de la Estructura
•	Tecnología: HTML5

•	Descripción:
o	El sitio web cuenta con una estructura clara y organizada que incluye:
	Página de inicio con productos destacados y ofertas.
	Catálogo de productos con filtros de búsqueda.
	Página de detalle de producto con información detallada y opciones de configuración.
	Carrito de compras donde los usuarios pueden revisar sus selecciones y proceder al pago.
o	La navegación es intuitiva, permitiendo a los clientes encontrar fácilmente los productos que están buscando.

2. Estilización con CSS
•	Tecnología: CSS3
•	Descripción:
o	Se aplica un CSS RESET para establecer un lienzo limpio y uniforme.
o	Los estilos CSS se personalizan para reflejar la identidad de la marca TechSpot, creando una experiencia visual atractiva y coherente.
o	El diseño es adaptativo, asegurando que el sitio se vea bien en dispositivos móviles, tabletas y computadoras de escritorio.

3. Integración de Bootstrap
•	Tecnología: Bootstrap 5
•	Descripción:
o	Bootstrap se utiliza para agilizar el desarrollo y garantizar la coherencia visual en todo el sitio.
o	Se implementan componentes de Bootstrap, como barras de navegación, tarjetas de producto y botones de llamada a la acción, para mejorar la usabilidad y la estética del sitio.

4. Desarrollo de Formularios
•	Tecnología: HTML5
•	Descripción:
o	Se desarrollan formularios para que los usuarios puedan registrarse, iniciar sesión y realizar compras.
o	Los formularios incluyen técnicas de validación de HTML5 para mejorar la precisión y seguridad de los datos ingresados.

Estructura del Proyecto
•	/assets: Contiene imágenes, fuentes y otros recursos multimedia.
•	/css: Contiene los archivos CSS personalizados y el archivo de CSS RESET.
•	/js: Contiene archivos JavaScript para mejorar la interacción y funcionalidad del sitio.
•	/index.html: Página de inicio del sitio web.
•	/catalog.html: Página de catálogo de productos.
•	/product.html: Página de detalle de producto.
•	/cart.html: Página de carrito de compras.
